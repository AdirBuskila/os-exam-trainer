דיסק ואחסון

נושא זה הוא חצי חשבון גאומטריה (נוסחה אחת שמסדרים מחדש) וחצי מושגים קלים (באפרינג: ספול ושמירה במטמון). דיסק קשיח מאחסן נתונים על דיסקיות מסתובבות בעלות גאומטריה פיזית קבועה, והתקנים איטיים זקוקים לבאפרינג כדי שה-CPU לא ימתין להם. כל שאלה היא או חשב מספר גאומטרי (צילינדרים, משטחים, דיסקיות, סך הסקטורים) או נקוב בשם מושג (היכן יושב ה-spool, אילו התקנים מדלגים עליו, מהו כיוון השמירה במטמון, ומהו המדיום הזול ביותר). דמיינו ערימת תקליטים: כל פאה היא משטח, טבעת אחת היא מסילה, אותה טבעת לאורך כל המשטחים היא צילינדר, וכל מסילה מחולקת לסקטורים.

שמירה במטמון (Caching)Cachingשמירה במטמוןהעתקת נתונים מאחסון איטי לאחסון מהיר יותר כדי להאיץ גישה חוזרת.
  1. כתבו את הנוסחה הראשית: קיבולת = צילינדרים × משטחים × סקטורים למסילה × בתים לסקטור.
  2. המירו כל גודל לחזקות של שתיים (1 KB = 2^10, 1 MB = 2^20, 1 GB = 2^30, 1 TB = 2^40; 0.5 TB = 2^39).
  3. הציבו את מה שידוע וסדרו מחדש כדי לפתור את הנעלם (חלקו את הקיבולת במכפלת הגורמים הידועים).
  4. החילו את שני הקשרים: משטחים = 2 × דיסקיות (דו-צדדית), ומסילות למשטח = מספר הצילינדרים.
  5. אם נשאלתם על סך הסקטורים, עצרו בצילינדרים × משטחים × סקטורים למסילה (בלי בתים לסקטור).
2020A Q2 - total sectors:
  1000 cylinders x 8 surfaces x 50 sectors/track = 400,000 sectors
  (total SECTORS -> stop here, do not multiply by bytes/sector)

2022A Q2 - cylinders (1TB, 2 double-sided platters, 2K spt, 4KB/sector):
  surfaces = 2 x 2 = 4 = 2^2 ; 2K = 2^11 ; 4KB = 2^12 ; 1TB = 2^40
  cylinders = 2^40 / (2^2 x 2^11 x 2^12) = 2^40 / 2^25 = 2^15 = 32K

2022B Q2 - platters (0.5TB, 8K cyl, 2K spt, 4KB/sector):
  0.5TB = 2^39 ; 8K = 2^13 ; 2K = 2^11 ; 4KB = 2^12
  surfaces = 2^39 / (2^13 x 2^11 x 2^12) = 2^39 / 2^36 = 2^3 = 8
  platters = 8 / 2 = 4

2023B Q2 - platters (1TB, 8K cyl, 8K spt, 8KB/sector):
  1TB = 2^40 ; 8K = 2^13 ; 8K = 2^13 ; 8KB = 2^13
  surfaces = 2^40 / (2^13 x 2^13 x 2^13) = 2^40 / 2^39 = 2^1 = 2
  platters = 2 / 2 = 1

2019A Q2 - tracks per surface (4000 cylinders, 2 platters):
  tracks/surface = number of cylinders = 4000
  (the platter count is a distractor)

2024B Q2מדיום האחסון הזול ביותרHDD
  • דיסקיות דו-צדדיות פירושן משטחים = 2 × דיסקיות. השאלה נותנת לעיתים קרובות דיסקיות בעוד שהנוסחה דורשת משטחים (או להפך) - הכפילו או חלקו ב-2 לפני ההצבה.
  • שמרו כל גודל בחזקות של שתיים. 0.5 TB = 2^39 (ולא 2^40), 2K = 2^11, 4 KB = 2^12. ערבוב עשרוני ובינארי הוא טעות החשבון הנפוצה ביותר.
  • מסילות למשטח = מספר הצילינדרים. מספר הדיסקיות שניתן לצידו הוא הסחת דעת.
  • "סך הסקטורים" עוצר בצילינדרים × משטחים × סקטורים למסילה - אל תוסיפו בתים לסקטור. "קיבולת" מכפילה גם בבתים לסקטור.
  • ספול נועד להתקנים איטיים והוא יושב על ה-HDD. התקן מהיר כמו מסך או מקלדת אינו זקוק לספול; הספול עניינו איטיות, לא קלט מול פלט.
  • כיוון השמירה במטמון הוא איטי -> מהיר (מעמידים עותק בשכבה המהירה). אל תהפכו את הכיוון.
  • ה-HDD הוא המדיום הזול ביותר בתשובת הקורס הזה.
