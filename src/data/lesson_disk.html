Disk & Storage

This topic is half geometry math (one formula, rearranged) and half easy concepts (buffering: spooling and caching). A hard disk stores data on spinning platters with a fixed physical geometry, and slow devices need buffering so the CPU does not wait on them. Every question is either compute a geometry number (cylinders, surfaces, platters, total sectors) or name a concept (where the spool sits, which devices skip it, the direction of caching, the cheapest medium). Picture a stack of vinyl records: each face is a surface, one ring is a track, the same ring across all surfaces is a cylinder, and each track is sliced into sectors.

  1. Write the master formula: capacity = cylinders × surfaces × sectors/track × bytes/sector.
  2. Convert every quantity to powers of two (1 KB = 2^10, 1 MB = 2^20, 1 GB = 2^30, 1 TB = 2^40; 0.5 TB = 2^39).
  3. Plug in what you know and rearrange to solve for the unknown (divide capacity by the product of the known factors).
  4. Apply the two relations: surfaces = 2 × platters (double-sided), and tracks per surface = number of cylinders.
  5. If asked for total sectors, stop at cylinders × surfaces × sectors/track (no bytes/sector).
2020A Q2 - total sectors:
  1000 cylinders x 8 surfaces x 50 sectors/track = 400,000 sectors
  (total SECTORS -> stop here, do not multiply by bytes/sector)

2022A Q2 - cylinders (1TB, 2 double-sided platters, 2K spt, 4KB/sector):
  surfaces = 2 x 2 = 4 = 2^2 ; 2K = 2^11 ; 4KB = 2^12 ; 1TB = 2^40
  cylinders = 2^40 / (2^2 x 2^11 x 2^12) = 2^40 / 2^25 = 2^15 = 32K

2022B Q2 - platters (0.5TB, 8K cyl, 2K spt, 4KB/sector):
  0.5TB = 2^39 ; 8K = 2^13 ; 2K = 2^11 ; 4KB = 2^12
  surfaces = 2^39 / (2^13 x 2^11 x 2^12) = 2^39 / 2^36 = 2^3 = 8
  platters = 8 / 2 = 4

2023B Q2 - platters (1TB, 8K cyl, 8K spt, 8KB/sector):
  1TB = 2^40 ; 8K = 2^13 ; 8K = 2^13 ; 8KB = 2^13
  surfaces = 2^40 / (2^13 x 2^13 x 2^13) = 2^40 / 2^39 = 2^1 = 2
  platters = 2 / 2 = 1

2019A Q2 - tracks per surface (4000 cylinders, 2 platters):
  tracks/surface = number of cylinders = 4000
  (the platter count is a distractor)

  • Double-sided platters means surfaces = 2 × platters. The question often gives platters while the formula needs surfaces (or vice versa) - multiply or divide by 2 before plugging in.
  • Keep every size in powers of two. 0.5 TB = 2^39 (not 2^40), 2K = 2^11, 4 KB = 2^12. Mixing decimal and binary is the number-one arithmetic slip.
  • Tracks per surface = number of cylinders. The platter count given alongside is a distractor.
  • "Total sectors" stops at cylinders × surfaces × sectors/track - do not tack on bytes/sector. "Capacity" multiplies by bytes/sector too.
  • Spooling is for slow devices on the HDD. A fast device such as a screen or keyboard needs no spool; spooling is about slowness, not input vs output.
  • Caching direction is slow -> fast (stage a copy in the faster tier). Do not reverse it.
  • The HDD is the cheapest medium in this course's answer.
